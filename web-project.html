Team 21

Homework 3: How websites work?

Purpose of the Project

The purposes of this project to provide hands-on experience in creating web content, to strengthen our coding abilities and understanding of web development basics, and to create a platform that demonstrates how HTML works, inspiring other students to start their web development journey.

What we did

Team 21 created this website. We designed a functional and user-friendly webpage to show how websites are built from scratch, using HTML and CSS. The project includes essential elements like:

  • Structuring content with HTML tags.
  • Organizing the layout with headings, paragraphs, lists, and tables.
  • Inserting multimedia such as images and videos.
  • Adding links for navigation and external resources.
  • Styling website using CSS.

Why We Chose HTML

We built this website using HTML because it is the backbone of web development. As SITE students, understanding HTML is crucial for:

  • Learning the fundamentals of how websites are structured.
  • Building a solid foundation before advancing to CSS, JavaScript, and backend technologies.
  • Showcasing how raw code transforms into a visual interface.

What we Learned

By creating this project, we learned:

  • Structuring websites effectively using semantic HTML.
  • Writing clean, organized, and reusable code.
  • Problem-solving challenges in coding and debugging.
  • The importance of planning and designing before implementation.
